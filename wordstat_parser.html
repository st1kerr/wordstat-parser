📖 Как получить Bearer Token и настроить прокси?
1. Прокси (одноразово):
  1. Откройте script.google.com → «Новый проект»
  2. Вставьте код прокси (файл proxy_script.gs)
  3. Развернуть → Новое развёртывание → Тип: Веб-приложение → Выполнять от: Меня → Доступ: Все → Развернуть
  4. Скопируйте URL вида https://script.google.com/macros/s/.../exec
2. Токен:
  1. Зайдите на oauth.yandex.ru → создайте приложение. Redirect URI → «Подставить URL для разработки». Запишите CLIENT_ID.
  2. Подайте заявку на API Вордстата: wordstat.yandex.ru → профиль → «Доступ к API» → укажите CLIENT_ID.
  3. Откройте: https://oauth.yandex.ru/authorize?response_type=token&client_id=ВАШ_CLIENT_ID
    В адресной строке скопируйте access_token — это Bearer Token.
